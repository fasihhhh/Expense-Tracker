# Expense-Tracker
A Full Stack Expense tracker that let you help in managing your earnings and deposits. Made with HTML, CSS, JavaScript ,Bootstrap and json server(json server for now).
Frontend: HTML, CSS/Bootstrap ,Javascript
Backend: Mock-APi implemented  using json-server
It also lets you download pdf of all the recent transcations.
Responsive Design: Optimized for both desktop and mobile devices
Charts: To show graphical representation of spendings.
